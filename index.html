VS2019如何添加include头文件目录、静态库、动态库

在编译《3d游戏编程大师技巧》raider3d项目时，由于DX11（我安装的dx版本）已经不再包含dmusici.h和dmusicf.h，所以需要手动导入dx8.0的SDK

添加头文件的顺序为：

  1. 下载解压好SDK包（csdn就是clm）

    里面有各种头文件和静态（lib）库文件

  2. 右键项目→属性→C/C++→常规→附加包含目录 中添加SDK文件夹

  3. OK 不过还有一堆bug去处理

    发现bug是由上面的操作引起的，不应该加到附加包含目录，在此程序中会影响winnt.h导致很多不容易修复的bug

    解决方法：

    在工程属性→配置属性→VC++目录→可执行文件目录 中添加

    注意：所添加的目录顺序必须在原目录的最后

    否则会因包含顺序不同导致编译顺序不同，造成bug，比如winnt.h中的未定义问题

添加静态库的顺序为：

  1. 在库目录中添加lib文件的路径

  1. 在下图位置输入要添加的lib文件名

添加动态库则比较简单，添加到资源目录即可

leetcode - 1013. 将数组分成和相等的三个部分

给你一个整数数组 A，只有可以将其划分为三个和相等的非空部分时才返回 true，否则返回 false。

形式上，如果可以找出索引 i+1 < j 且满足 (A[0] + A[1] + … + A[i] == A[i+1] + A[i+2] + … + A[j-1] == A[j] + A[j-1] + … + A[A.length - 1]) 就可以将数组三等分。

**

示例 1：

输出：[0,2,1,-6,6,-7,9,1,2,0,1]
输出：true
解释：0 + 2 + 1 = -6 + 6 - 7 + 9 + 1 = 2 + 0 + 1
示例 2：

输入：[0,2,1,-6,6,7,9,-1,2,0,1]
输出：false
示例 3：

输入：[3,3,6,5,-2,2,5,1,-9,4]
输出：true
解释：3 + 3 = 6 = 5 - 2 + 2 + 5 + 1 - 9 + 4

提示：

3 <= A.length <= 50000
-10^4 <= A[i] <= 10^4

一、思路

  • 其实已知每份的和，不用试探着去找
  • 注意是非空部分，也就是各部分得相等还要保证至少有一个值

求出每份的和sum，用前后指针求该和，计算中间和是否等于两边

注意数组指针：

  • 越界问题，l < h
  • 中间值为一个的情况，l == h别忘了
