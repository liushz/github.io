    auto carry = ((unsigned int )(a & b))<<1;

leetcode.367 有效的完全平方数

给定一个正整数 num，编写一个函数，如果 num 是一个完全平方数，则返回 True，否则返回 False。

说明：不要使用任何内置的库函数，如 sqrt。

示例 1：

输入：16
输出：True
示例 2：

输入：14
输出：False

一、思路

三种方法：

  1. 暴力找法：从1开始，将其平方与num比较，直至等于（true）或者大于（false）num

    注意：平方和平方的基数都得用long，最好long long，因为可能平方超int最大值 2^31 -1

  2. 二分查找法：从2~num二分找

  3. 牛顿迭代法

  • 设方程f（x） = num - x0^2 = 0

  • 给一个初始 x0 = num/2,比较其平方和num

  • $$
    易知 f(x0)’ = f(x0)/(x1 - x0)

    f(x0) = num - x0^2

    f(x0)’ = 2x0

    得 x1 = (num + x0^2 )/ 2x0
    $$

3的幂

我采用的是最基础的方法，循环比较%3是否为0，除3，判断是否为1

class Solution { public: bool isPowerOfThree(int n) { if(n == 1) return true; else if (n < 1) return false; if(n % 3 !=0) return false; n/=3; return isPowerOfThree(n); } };
