第一部分Cpp基础

一、Cpp基本数据类型

1.1、 基本数据类型

Cpp提供：整形、浮点数、字符、bool和void类型

  • 整形int为主，推荐用int
  • 浮点数，有单精度浮点数float和双精度浮点数double，推荐double
  • 字符，8位，以便存储基本字符？
  • bool void 不加赘述

1.2 、数据类型相互转换

  • 浮点→整形会舍弃小数点后
  • 很大的整形→浮点会丧失精度
  • unsigned超出范围后，取模：8位unsigned范围0 ~ 255 ，-1取255 （-1+256）%256

1.3 、初始化

  • 函数体内变量（对象object）初始化为“未定义”，初始化后用

  • 函数体外变量初始化为“0”，据数据类型不同0不同

  • 建议都初始化，易出错且难改

1.4 、 声明、定义、初始化

  • extern int i ;//声明非定义，一个文件想使用别处定义的变量需在本文件内声明
  • int i ；//声明，定义 （创造一个实体整形对象）

1.5 、变量标识符

  • 只能由字母、下划线、数字组成

  • 由字母 or 下划线开头

  • 为cpp保留的标识符不能用

  • cpp标准库保留：

    • 不能连续两个下划线
    • 不能下划线紧连大写字母开头
    • 函数体外 的标识符不能以下划线开头

    命名规范

  1. 标识符体现实际意义 ，如：student_number
  2. 变量名一般用小写字母 ，如：index
  3. 自定义类名一般大写字母开头，如：Sales_item
  4. 多单词分割
    • student_loan
    • studentLoan

1.6 引用

引用就是对象的别名 int &rename = name;

  • 引用必须被初始化，因为它和对象是绑定关系，一旦绑定，无法重新绑定
  • 引用不是一个对象，不能定义引用的引用

1.7 指针

指针指向地址，解引用后得到其对象 int *p = &var;

  • 指针本身是一个对象
  • 不能定义指向引用的指针,引用不是个对象，没有实际地址

指针指向的情况

  1. 对象
  2. 对象空间的下一个位置
  3. 空指针，未指向任何一个对象（未初始化或者 nullptr，预处理变量NULL）
  4. 无效指针，上述情况外的值

void* 类型，可存任意类型对象的地址

离变量名最近的符号对变量类型有直接的影响，如：int *&r = p； // r 是一个对指针的引用

1.7 const

常量constant

  1. 必须初始化

  2. 对const的引用 == 常量引用，引用是无法改变绑定对象的

    • 对常量对象只能用常量引用，普通引用会报错

    • 一般引用只能绑定对象，而常量引用可用任意表达式作为初始值
