《3D游戏编程大师技巧》 - 环境配置

第一章的小Demo raider3D的配置，游戏文件结果为：

主循环cpp + 游戏库cpp和h + DirectX的lib →（编译 + dll）exe

问题如下：

  1. 错误 C2664 “int Open_Error_File(char *,FILE *)”: 无法将参数 1 从“const char [10]”转换为“char *”

    解决方法：项目属性→ C/C++->语言->符合模式 →否

  1. 错误 C1083 无法打开包括文件: “dmusici.h”，“dmusici.h”: No such file or directory

    解决方法：

    • 要么安装dx8.0 or 9.0 （我安装了没解决，应该是我已经有高本版的问题）
    • 要么用dx8.0的SDK，万恶的csdn有，下了后包含.h、.lib、dll，包含方法

  2. This function or variable may be unsafe. Consider using strtok_s instead. To disable deprecation, use _CRT_SECURE_NO_WARNINGS. See online help for details.

    如图加上_CRT_SECURE_NO_WARNINGS；

  3. 不能将 “const char *” 类型的值分配到 “LPCWSTR” 类型的实体

    更改：项目属性→配置属性→高级→字符集 改为“使用多字节字符集”
