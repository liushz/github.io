3的幂

给定一个整数，写一个函数来判断它是否是 3 的幂次方。

我采用的是最基础的方法，循环比较%3是否为0，除3，判断是否为1

class Solution { public: bool isPowerOfThree(int n) { if(n == 1) return true; else if (n < 1) return false; if(n % 3 !=0) return false; n/=3; return isPowerOfThree(n); } };

注意1是3的最小次幂，0次

时间空间都一般般，中规中矩

其他方法：（leetcode官方题解）

  • 最简单的，int32位，最大的3次幂为 2147483647 （复习一下，int最大值 2^31 - 1)，然后用它对n取余为0的是正确结果

  • 较奇妙的：例如10进制：1000000，将其转化为3进制形式字符串，如27=3^3 =1000(3)

    再去正则匹配，java两个函数就解决了，cpp我研究研究：

    *c++有正则匹配库 https://www.cnblogs.com/coolcpp/p/cpp-regex.html *以后用到再研究

    以上都要注意3的幂不会小于1
