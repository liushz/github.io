leetcode.371 两整数和

不使用运算符 + 和 - ，计算两整数 a 、b 之和。

示例 1:

输入: a = 1, b = 2
输出: 3
示例 2:

输入: a = -2, b = 3
输出: 1

一、思路

一开始想不出来，但隐约觉得是用二进制实现加法，看了题解果然是，二进制加法是组原学的，学的一知半解，正好复习一下。

加法拆分两步：

  1. a b取亦或^（第一次见cpp亦或，惭愧-。-），实现 不进位的相加和

    1 1 → 0 （进位）

    0 0 → 0

    1 0 →1

    0 1 →1

    如 3 9 ： 0011 ^ 1001 → 1010 （最后一位产生进位）

  2. a b 取和& ，实现进位

    如：0001 & 0001 → 0001 末位进位1

    此时知道哪位进位了，但进位方向是向前一位，所以左移

    auto carry = ((unsigned int )(a & b))<<1;

此时相当于得到了新的两个数ab，相加其得结果（不进位和+进位），那就重复上述步骤呗

直至进位为0，就是和
